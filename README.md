# AD Producto

## Descripción
Este proyecto es un sitio web estático desarrollado para promocionar y proporcionar información sobre el producto AD. Incluye secciones como inicio, eventos, noticias, ponentes, patrocinadores y contacto.

## Autor
Antony Jampol Aquino Rumualdo

## Instalación
No se requiere instalación especial. Simplemente clona el repositorio y abre los archivos HTML en tu navegador web.

```bash
git clone https://github.com/tu-usuario/AD_producto.git
cd AD_producto
# Abre index.html en tu navegador
```

## Uso
- Navega por las diferentes páginas HTML para explorar el contenido.
- Las páginas incluyen: `index.html` (inicio), `eventos.html`, `noticias.html`, `ponentes.html`, `patrocinadores.html` y `contacto.html`.
- Los estilos están definidos en `css/styles.css`.
- La funcionalidad interactiva se maneja con `js/script.js`.

## Estructura del Proyecto

## Tecnologías Utilizadas
- **HTML5**: Estructura del sitio web
- **CSS3**: Estilos y diseño
