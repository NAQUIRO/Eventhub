# EventHub | Portal de Congresos y Seminarios

## Descripción
**EventHub** es una plataforma web estática dedicada a la difusión y promoción de congresos, seminarios y eventos académicos a nivel nacional e internacional. Su objetivo es conectar a estudiantes, profesionales e instituciones en un espacio digital donde el conocimiento y la innovación se encuentren.

El proyecto permite a los usuarios:
*   Explorar próximos eventos académicos.
*   Conocer a los ponentes destacados.
*   Informarse sobre noticias relevantes.
*   Ver las empresas patrocinadoras.
*   Contactar con los organizadores.

## Características Principales
*   **Inicio**: Visión general y destacados del portal.
*   **Eventos**: Catálogo de congresos y seminarios con detalles (fechas, ubicación).
*   **Ponentes**: Información sobre los expertos que participan en los eventos.
*   **Patrocinadores**: Espacio para las entidades que apoyan los eventos.
*   **Noticias**: Novedades y actualizaciones del mundo académico.
*   **Contacto**: Formulario e información para comunicarse con la administración.

## Tecnologías Utilizadas
*   **HTML5**: Para la estructura semántica del contenido.
*   **CSS3**: Para el diseño visual, estilos y responsividad (`css/styles.css`).
*   **JavaScript**: Para la lógica y funcionalidad interactiva (`js/script.js`).

## Instalación y Uso
No se requiere ninguna instalación de dependencias del lado del servidor.

1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com/tu-usuario/AD_producto.git
    cd AD_producto
    ```

2.  **Ejecutar:**
    Simplemente abre el archivo `index.html` en tu navegador web preferido (Chrome, Firefox, Edge, etc.).
